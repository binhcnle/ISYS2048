To setup the app please follow these steps
1/ Install XAMPP
2/ Start XAMPP Apache Server and MySQL Server
3/ Go to http://localhost/ then choose phpMyAdmin
4/ Import the education.sql
5/ Go to XAMPP installed location, then go to htdocs
6/ Copy to 2 folder "education_api" and "education_app"
7/ To see JSON Data, Open your web browser and go to http://localhost/education_api/university.php
8/ To see how you app retrive that information, go to http://localhost/education_app/get_all_university.html
